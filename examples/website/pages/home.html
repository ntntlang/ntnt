Correct by Construction

In NTNT, you don't just write code; you specify behavior. Adding contracts to your functions ensures they always behave as intended.

fn withdraw(account: BankAccount, amount: Int) -> Bool
    requires amount > 0
    ensures result == true implies account.balance >= 0
{
    if account.balance >= amount {
        account.balance = account.balance - amount
        return true
    }
    return false
}

Get Started in Seconds

Get up and running with a single command. The NTNT compiler is fast, lightweight, and has zero dependencies.

# Clone the repository
git clone https://github.com/ntntlang/ntnt.git
cd ntnt

# Build and run your first program
cargo build --release
./target/release/ntnt run examples/hello.tnt
