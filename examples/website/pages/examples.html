Contracts

fn divide(a: Int, b: Int) -> Int
    requires b != 0  // Precondition
    ensures result * b == a  // Postcondition
{
    return a / b
}

HTTP Server

import { json } from "std/http/server"

fn get_status(req) {
    return json(map {
        "status": "ok",
        "version": "0.1.6"
    })
}

get("/api/status", get_status)
listen(3000)

Running Examples

To run any example, use the ntnt CLI directly. It is fast, lightweight, and handles dependencies automatically.

1
Select an Example

Choose any .tnt file from the examples/ directory of the repository.

2
Run the Command

Use ntnt run for scripts or servers, or ntnt test for verifying API endpoints.

3
Observe Results

The NTNT runtime will execute the code and automatically validate all contracts and invariants.

# Run a script
ntnt run examples/hello.tnt

# Run a server
ntnt run examples/http_server.tnt

# Test an endpoint
ntnt test examples/http_server.tnt --get /api/status

# Test an endpoint with a JSON payload
ntnt test examples/http_server.tnt --post /users --body '{"name":"Alice"}'

# Verbose output (shows headers)
ntnt test examples/http_server.tnt --get /api/status --verbose
