Introducing NTNT: A Programming Language for the AI Era

January 15, 2026 - NTNT Team

We're excited to introduce NTNT (pronounced "Intent"), a new programming language designed from the ground up for AI-assisted software development.

A Real Programming Language

NTNT is not a toy or a DSL. It's a full-featured language with everything you need to build real applications:

  • Modern type system - Generics, enums, pattern matching, structs, and traits
  • Design by contract - requires and ensures clauses built into the syntax
  • Comprehensive stdlib - HTTP server/client, PostgreSQL, JSON, CSV, file I/O, crypto, and concurrency
  • No package manager needed - Batteries included, just import and use

What Makes NTNT Different?

Traditional languages were designed for humans typing code character by character. NTNT is designed for a world where AI agents generate code and humans verify it works correctly.

The key innovation is Intent-Driven Development (IDD):

  1. Write requirements in plain language .intent files
  2. Link code to requirements with @implements annotations
  3. Run ntnt intent check to verify they match

This creates full traceability from requirement to implementation. You can always answer: "What is this code supposed to do?" and "Does it actually do that?"

Developer Experience

We've focused on making NTNT a joy to use:

  • Native hot-reload - HTTP servers automatically reload when you save. No restart needed.
  • Intent Studio - Visual development environment with live test execution
  • Single binary - No runtime dependencies, runs anywhere
  • Agent-native tooling - ntnt inspect outputs structured JSON for AI agents

Who Should Use NTNT?

NTNT is experimental and not production-ready. It's a great fit for:

  • Exploring AI-assisted development workflows
  • Prototypes and proof-of-concepts
  • Learning about contracts and formal specifications
  • Internal tools where you control the environment
