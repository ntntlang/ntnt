# NTNT Language Source Code

This directory contains the implementation of the NTNT programming language.

## Structure

- `compiler/` - Language compiler and type checker
- `runtime/` - Virtual machine and execution engine
- `stdlib/` - Standard library implementation
- `src/` - Core language source files
- `examples/` - Sample programs and use cases
- `tests/` - Test suites and specifications
- `docs/` - Documentation and specifications
