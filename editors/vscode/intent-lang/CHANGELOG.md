# Change Log

All notable changes to the Intent Language extension will be documented in this file.

## [0.1.0] - 2026-01-05

### Added

- Initial release
- Syntax highlighting for `.intent` and `.itn` files
- Language configuration (comments, brackets, folding)
- Code snippets for common patterns:
  - Functions with and without contracts
  - Structs and impl blocks
  - Control flow statements
  - Contract clauses (requires, ensures, invariant)
- Highlighting for:
  - Keywords and control flow
  - Contract keywords (requires, ensures, invariant, old, result)
  - Effect keywords (pure, io, network, etc.)
  - Primitive and collection types
  - Built-in functions
  - Comments (line and block)
  - Strings (double, single, template)
  - Numbers (integer, float, hex, binary, octal)
  - Operators
